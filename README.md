# CustomTrial

---

## 中文版本

**CustomTrial** 是一个使用 **Pathfinder 模组框架** 的 Hacknet 模组，用于创建自定义试炼程序。

> ⚠️ 目前本项目仅是一个空壳，尚未实现具体功能。

---

## English Version

**CustomTrial** is a Hacknet mod using the **Pathfinder mod framework**, designed to create custom trial programs.

> ⚠️ Currently this project is just an empty shell; no actual features have been implemented yet.

---

## 备注 / Notes

- 本模组基于 Hacknet 游戏开发。
- Pathfinder 框架提供了必要的 API 支持。
- 欢迎后续贡献代码或提出建议。
